# 박도겸 캐릭터 에셋

참조 페르소나: ../../02_character_persona/박도겸.md

## 핵심 이미지 방향
- 50대 후반~60대 초반 남성, 폐역 야간 경비원.
- 오래 무서워해서 무뎌진 생존자. 평온한 얼굴과 비정상적 상황의 대비가 중요.
- 기본 의상은 낡은 경비복, 손전등, 열쇠, 순찰 기록표, 무전기.
- 시각 요소는 폐역 조명, 반복 기록, 무전 잡음, 2:17 시계.

## 유지해야 할 요소
- 호들갑 없는 무덤덤함.
- 피로와 죄책감이 배어 있는 얼굴.
- 오래된 기록과 열쇠를 들고 있는 손.

## 금지 요소
- 괴담 해설자 같은 과장 표정.
- 용감한 액션 경비원.
- 코믹한 노인 캐릭터화.
