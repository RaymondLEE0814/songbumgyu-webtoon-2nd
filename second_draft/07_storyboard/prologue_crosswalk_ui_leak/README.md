# 프롤로그 스토리보드: 횡단보도 UI 누출

밝은 출근길 사거리에서 한서윤만 현실의 프레임 지연과 `02:17` 오류를 감지하는 프롤로그입니다. 기존 EP01의 밤 지하철 체험을 본편으로 남겨두기보다, 프롤로그에서는 더 일상적이고 밝은 표면에서 세계관의 첫 균열만 보여줍니다.

## 원고
- 원고 초안: `../../05_episode_draft/prologue_draft.md`

## 핵심 연출
- 어둠, 폐역, 귀신 형상으로 시작하지 않는다.
- 출근길의 밝음, 질서, 반복 리듬을 먼저 충분히 보여준다.
- 이상현상은 한 사람의 프레임 지연, 신호등 숫자의 `02:17`, 햇빛 속 픽셀 먼지로 제한한다.
- 승차권 조각, 박도겸 문자, AI 해석은 아직 쓰지 않는다.

## 프롤로그가 남기는 질문
- 서윤은 왜 이 현상을 처음 겪는 사람처럼 반응하지 않는가?
- `02:17`은 시간인가, 신호인가, 도착하지 않은 무언가의 표시인가?
- 밝은 현실에서도 새는 정보라면, 폐역에서는 무엇이 더 많이 드러나는가?

## 현재 진행 상태
- 프롤로그 원고 초안 작성 완료.
- 프롤로그 스토리보드 상세화 완료.
- 제작용 컷 리스트 업데이트 완료.
- 작화/연출 제작 노트 업데이트 완료.
- 프롤로그 세로 스크롤 `layout_draft` SVG/HTML/TSV 생성 완료.
- 아직 최종 작화용 이미지 러프와 검증 문서는 만들지 않았다.

## 포함 문서
- `prologue_storyboard.md`: 컷별 목적, 구도, 화면, 효과, 다음 컷 연결까지 정리한 상세 스토리보드.
- `panel_list.md`: 제작용 컷 리스트와 이미지화 우선 컷.
- `production_notes.md`: 작화/연출 주의사항 및 다음 작업 순서.
- `layout_draft/`: 프롤로그 세로 스크롤 레이아웃 초안.
