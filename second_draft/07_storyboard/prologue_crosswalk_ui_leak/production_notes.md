# 프롤로그 제작 노트

## 톤
- 공포의 출발점을 밤, 어둠, 지하 공간으로 잡지 않는다.
- 햇빛, 유리 반사, 신호등 LED, 카페 간판처럼 밝고 선명한 요소를 전면에 둔다.
- 이상현상은 밝은 화면 위의 렌더링 오류처럼 보여야 한다.
- 도시가 무섭게 생긴 것이 아니라, 도시의 표시 체계가 아주 잠깐 틀리는 쪽이 핵심이다.

## 서윤 연기
- 비명을 지르거나 도망치지 않는다.
- 먼저 몸이 반응하고, 그 다음 확인하려 한다.
- "또 안 찍혔네", "샜어"처럼 처음이 아니라는 뉘앙스를 짧게 남긴다.
- 감정은 얼굴보다 행동으로 보인다: 멈춤, 손목 움켜쥠, 카메라 확인, 숫자 재확인.

## 노이즈 표현
- 회색 재킷 남자는 귀신이나 괴물처럼 디자인하지 않는다.
- 지연된 프레임, 밀린 윤곽선, 햇빛 속 픽셀 먼지 정도로 제한한다.
- `02:17`은 아주 짧게 박아 넣고 설명하지 않는다.
- RGB 글리치가 과하면 사이버펑크처럼 보일 수 있으니, 프롤로그에서는 미세하고 현실적인 오류로 유지한다.

## 아껴둘 요소
- 승차권 조각은 프롤로그에서 쓰지 않는다.
- 박도겸의 경고 문자는 공개하지 않는다.
- 윤태하와 AI는 등장시키지 않는다.
- 폐역명 전체는 공개하지 않는다.
