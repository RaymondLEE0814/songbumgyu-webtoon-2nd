# 한서윤의 오빠 에셋 일관성 체크

## 통과 기준
- 보호하려는 절박함이 보이는가?
- 죽은 유령보다 "아직 도착하지 않은 사람"처럼 느껴지는가?
- 1999년 학생의 시대감이 과하지 않게 드러나는가?
- 승차권 반쪽과 시간 간섭 이미지가 연결되는가?

## 수정 필요 기준
- 천사/유령처럼 편안하게 보인다.
- 히어로처럼 선명하고 강해 보인다.
- 악의적인 존재처럼 보인다.
- 서윤을 지키려는 감정이 사라진다.
