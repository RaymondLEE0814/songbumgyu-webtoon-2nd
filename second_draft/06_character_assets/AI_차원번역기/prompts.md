## 01_interface_concept
Use case: ui-mockup
Asset type: webtoon AI system visual reference
Primary request: Create a visual concept sheet for an AI noise translation system that reads discarded human perception noise as dimensional communication.
Scene/backdrop: dark workstation screen environment, no human character.
Subject: abstract AI interface with CCTV frames, waveform strips, subway announcement audio patterns, talisman-like line structures, coordinate markers, corrupted blocks; output feels incomplete and unsettling.
Style/medium: grounded webtoon production design, interface concept art, analog-digital horror.
Composition/framing: main monitor UI concept, close-up panels for waveform, CCTV noise block, talisman-code overlay, output strip.
Lighting/mood: cold monitor glow, clinical dread, non-personified.
Color palette: black, cold gray, desaturated teal, off-white scanlines, small red warning accents.
Constraints: no cute robot, no face, no readable long text; visual information should be abstract and reusable.
Avoid: humanoid AI avatar, friendly chatbot UI, fantasy magic circle, excessive neon cyberpunk, watermark.
