# 메인 스토리 플롯 (테스트 연재 기준)

'인지-차원-무속' 세계관을 빠르고 가볍게 독자에게 전달하기 위한 초반부 플롯 구조입니다.

## 로그라인
일상의 노이즈와 환각에 시달리며 '차원 필터'가 무너져가는 무당과, 그 노이즈를 오류가 아닌 '통신'으로 해석하는 AI 개발자가 만나 세계의 진짜 껍질을 벗겨내는 이야기.

## 보완 로그라인
1999년 폐역 실종 사건을 취재하던 구성작가 한서윤은 "귀신"이라 불리던 것이 죽은 사람의 영혼이 아니라, 인간의 인지 필터 밖에서 새어 나온 정보 구조임을 알게 된다. 그 정보를 AI로 번역하려는 윤태하와, 20년 동안 같은 현상을 관측해온 경비원 박도겸이 얽히며 서윤은 오빠의 실종이 아직 끝나지 않은 사건이라는 사실에 접근한다.

## 1막 핵심 질문
- 오빠는 죽은 것인가, 아직 도착하지 않은 것인가?
- 귀신은 사람인가, 사건인가, 통신인가?
- AI가 읽는 노이즈는 진실인가, 고차원 정보의 오역인가?
- 서윤이 기억을 되찾으면 사건은 해결되는가, 다시 시작되는가?

## 프롤로그
### 프롤로그: 횡단보도 UI 누출
- 밝은 출근길 사거리에서 서윤은 사람들 사이에 섞여 회의 장소로 향합니다.
- 보행자 신호등 숫자가 정상적으로 줄어들다가 순간 `02:17`로 고정되고, 인파 속 한 사람만 프레임이 한 박자 늦게 따라오는 것처럼 보입니다.
- 서윤은 처음 겪는 사람처럼 놀라지 않고, 손목의 픽셀 무늬와 이명으로 "또 샜다"는 사실을 알아챕니다.
- 휴대폰 카메라에는 아무것도 찍히지 않지만, 직후 도착한 파일 `폐역_제보영상_1999.mp4`의 썸네일에 `02:17`이 희미하게 깜빡입니다.
- 독자는 세계관 설명 없이, 밝은 현실에서도 무언가 새어 나오며 그 현상이 1999년 폐역과 연결된다는 감각만 받습니다.

## 초반부 플롯 (1~5화)
### 1화: 필터의 붕괴 (체험의 단계)
- 프롤로그에서 도착한 `폐역_제보영상_1999.mp4`를 방송국 회의실에서 재생하며 시작합니다.
- 폐역 CCTV 속 `02:17` 오류와 프롤로그의 횡단보도 프레임 지연이 같은 패턴임을 서윤만 알아차립니다.
- 밝은 골든아워 지상철에서 현상이 더 강하게 재발하고, 맞은편 승객의 햇빛 닿은 윤곽이 2D 바코드와 하얀 픽셀 입자로 풀립니다.
- 마지막에 바닥으로 스며든 픽셀 잔해가 오래된 승차권 조각으로 남고, 저장명 없는 번호가 "그쪽도 기록됩니다"라는 경고 문자를 보냅니다.
- 독자는 세계관 설명 없이, 밝은 현실의 작은 오류가 폐역 사건과 직접 연결되어 있다는 사실만 압니다.

### 2화: 해석가의 등장
- 시점이 바뀌어, 다른 인물(AI 개발자)의 건조한 일상.
- 윤태하는 영상/음성 노이즈를 처리하는 모델을 훈련 중입니다.
- 모델은 백색 소음 속에서 "2:17", "도착하지 않음", "관측자 교체" 같은 짧은 문장을 반복 출력합니다.
- 태하는 이 출력이 폐역 CCTV의 깨짐, 전통 부적의 획, 지하철 안내방송의 파형과 같은 구조라는 사실을 확인합니다.

### 3화: 접점 (무당과 AI)
- 서윤은 취재팀과 함께 폐역에 들어가고, 박도겸은 20년 치 순찰 기록을 감추려 합니다.
- 새벽 2시 17분, 서윤의 몸이 먼저 반응하고 태하의 AI가 같은 순간을 원격으로 감지합니다.
- 두 사람은 처음 만난 자리에서 서로를 믿지 않지만, 같은 노이즈를 전혀 다른 방식으로 읽었다는 점 때문에 협력할 수밖에 없습니다.

### 4~5화: 노이즈의 정체
- AI가 번역한 노이즈의 텍스트가 서윤의 잃어버린 기억과 일치합니다.
- 귀신이나 저주가 아니라, 누군가 혹은 무언가가 붕괴된 통신 채널을 통해 이쪽을 계속 관측하고 있다는 단면이 드러납니다.
- 1막 종료 시점에서 서윤은 오빠가 "죽은 사람"이 아니라 "아직 도착하지 않은 정보 구조"일 수 있다는 가능성을 봅니다.

## 초반부 감정 곡선
- **서윤:** 부정 -> 신체 반응 -> 증거 집착 -> 기억의 균열.
- **태하:** 가설 확신 -> AI 출력 과신 -> 서윤이라는 변수를 만남 -> 첫 가설 수정.
- **도겸:** 회피 -> 방조 -> 작은 증언 -> 관측자로서의 죄책감.

## 회차별 설명량 제한
- 1화: 세계관 설명 금지. 오직 체험과 단서.
- 2화: AI가 노이즈를 패턴으로 본다는 사실만 설명.
- 3화: 무속과 AI가 같은 현상을 다른 언어로 읽는다는 대비만 제시.
- 4~5화: 귀신이 "잔류 정보 구조"일 수 있다는 가설을 처음으로 언어화.
