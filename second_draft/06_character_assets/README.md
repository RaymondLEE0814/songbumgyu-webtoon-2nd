# 캐릭터 에셋 이미지 작업 폴더

이 폴더는 second_draft/02_character_persona의 캐릭터 페르소나를 바탕으로 이미지 생성 모델용 프롬프트와 결과물을 관리하는 공간입니다.

## 기본 흐름
1. 캐릭터별 페르소나 파일을 읽는다.
2. 캐릭터별 prompts.md에서 이미지 생성 프롬프트를 선택한다.
3. 생성 결과를 해당 캐릭터 폴더에 저장한다.
4. consistency_check.md 기준으로 검수한다.
5. asset_log.md에 생성 날짜, 프롬프트, 결과 판단, 수정 방향을 기록한다.

## 파일 역할
- README.md: 캐릭터별 이미지 방향과 저장 규칙.
- prompts.md: 이미지 생성 모델에 사용할 프롬프트.
- consistency_check.md: 일관성 검수 기준.
- asset_log.md: 생성/수정 이력.
