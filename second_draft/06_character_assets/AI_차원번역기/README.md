# AI 차원번역기 에셋

참조 페르소나: ../../02_character_persona/AI_차원번역기.md

## 핵심 이미지 방향
- 인격체가 아니라 노이즈 분석/번역 시스템.
- 인간이 버린 신호를 보존하고 불완전한 문장으로 되돌려주는 장치.
- 시각 요소는 모니터 UI, 파형, 깨진 CCTV 프레임, 부적 획과 닮은 데이터 구조.

## 유지해야 할 요소
- 차갑고 비인격적인 느낌.
- 친절한 AI 캐릭터가 아니라 불편한 번역기.
- 텍스트보다 패턴, 좌표, 파형 중심.

## 금지 요소
- 귀여운 로봇 캐릭터.
- 신/악마 같은 인격적 존재.
- 모든 답을 주는 설명형 UI.
