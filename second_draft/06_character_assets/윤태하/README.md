# 윤태하 캐릭터 에셋

참조 페르소나: ../../02_character_persona/윤태하.md

## 핵심 이미지 방향
- 30대 중반 남성, 전 양자정보 연구자이자 독립 분석가.
- 차갑고 논리적인 인상, 수면 부족과 죄책감이 얇게 배어 있음.
- 기본 의상은 어두운 셔츠, 실험실 코트가 아닌 실용적 재킷, 노트북/태블릿/센서 장비.
- 시각 요소는 데이터 노이즈, 파형, CCTV 프레임, 부적 획과 겹치는 코드 구조.

## 유지해야 할 요소
- 감정보다 분석이 먼저인 자세.
- 장비를 다루는 손, 관찰하는 눈.
- AI 출력에 대한 신뢰와 불안이 동시에 있는 분위기.

## 금지 요소
- 미친 과학자 과장.
- 미래형 사이버펑크 과장.
- 퇴마사나 무속인처럼 보이는 의상.
