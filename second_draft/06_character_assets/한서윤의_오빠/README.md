# 한서윤의 오빠 캐릭터 에셋

참조 페르소나: ../../02_character_persona/한서윤의_오빠.md

## 핵심 이미지 방향
- 1999년 실종 당시 10대 후반 남학생.
- 죽은 사람이 아니라 아직 도착하지 못한 정보 구조.
- 보호하려는 절박함, 경고와 은폐의 모순.
- 시각 요소는 오래된 교복/사복, 찢어진 승차권의 반쪽, 지하철 선로, 시간 간섭 잔상.

## 유지해야 할 요소
- 동생을 보호하려는 다급함.
- 현재 시점에 완전히 존재하지 못하는 흐릿함.
- 따뜻함과 위험 경고가 함께 느껴지는 이미지.

## 금지 요소
- 편하게 등장하는 유령 오빠.
- 선명한 히어로 이미지.
- 모든 답을 아는 안내자처럼 보이는 연출.
