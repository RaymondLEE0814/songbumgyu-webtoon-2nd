# EP01 스토리보드: 필터의 누출

## 목적

## 회차 핵심
- 서윤은 폐역 취재 전날 지하철에서 노이즈 누출을 겪는다.
- 맞은편 승객은 괴물이 아니라 해석 불가능한 정보 구조처럼 무너진다.
- 노이즈 잔해는 오래된 승차권 조각으로 남는다.
- 박도겸의 경고 문자가 다음 화의 폐역 진입을 예고한다.
