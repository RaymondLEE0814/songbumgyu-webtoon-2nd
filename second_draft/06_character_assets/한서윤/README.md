# 한서윤 캐릭터 에셋

참조 페르소나: ../../02_character_persona/한서윤.md

## 핵심 이미지 방향
- 30대 초반 여성, 방송국 미스터리 프로그램 프리랜서 구성작가.
- 차분하고 지친 인상, 공포보다 관찰과 분석이 먼저 나오는 얼굴.
- 기본 의상은 어두운 실용적 코트, 셔츠나 니트, 취재용 가방, 자료 봉투.
- 시각 요소는 지하철 형광등, CCTV 노이즈, 승차권 조각, 손목의 픽셀 무늬.

## 유지해야 할 요소
- 건조하고 직설적인 눈빛.
- 피로하지만 무너지지 않는 자세.
- 오빠 단서 앞에서만 흔들리는 억제된 감정.

## 금지 요소
- 전형적인 퇴마사, 무당복, 과도한 판타지 의상.
- 공포에 질린 피해자 표정만 반복.
- 화려한 액션 포즈.
