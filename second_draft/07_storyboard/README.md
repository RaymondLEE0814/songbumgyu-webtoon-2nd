# 스토리보드 작업 폴더

이 폴더는 `.agents/work_flow.md` 기준으로 회차별 원고를 실제 제작 가능한 스토리보드 문서로 정리하는 공간입니다.

## 작업 흐름
1. `05_episode_draft`의 회차 원고를 가져온다.
2. `02_character_persona`와 `06_character_assets`를 확인해 인물 일관성을 맞춘다.
3. `03_art_direction`의 컷 리듬과 시각 스타일을 컷별로 반영한다.
4. 회차별 폴더에 스토리보드, 컷 리스트, 제작 노트, 검증 문서를 만든다.
5. 검증 후 최종 원고/이미지 제작 단계로 넘긴다.
