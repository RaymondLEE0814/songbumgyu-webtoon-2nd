# 이름 없는 여자 캐릭터 에셋

참조 페르소나: ../../02_character_persona/이름없는여자.md

## 핵심 이미지 방향
- 1999년 사건과 연결된 여고생 형상의 잔류 정보.
- 전형적 귀신이 아니라, 특정 순간이 사람 모양으로 낮게 번역된 상태.
- 젖은 교복, 찢어진 승차권, 정지된 자세, 고정된 시선 부재.
- 시각 요소는 픽셀 깨짐, 2D 바코드, CCTV 고정 프레임, 물방울이 멈춘 듯한 질감.

## 유지해야 할 요소
- 움직이지 않는 정적 공포.
- 감정 표현보다 반복 규칙과 미세한 오류.
- 사람과 정보 구조 사이의 불완전한 상태.

## 금지 요소
- 원혼처럼 공격하거나 추격하는 이미지.
- 긴 머리 귀신 클리셰 과장.
- 피투성이 점프 스케어.
