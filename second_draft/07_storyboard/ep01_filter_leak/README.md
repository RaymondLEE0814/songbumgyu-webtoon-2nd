# EP01 스토리보드: 필터의 누출

## 목적
1화는 프롤로그의 `폐역_제보영상_1999.mp4` 알림을 실제 사건으로 연결하고, 한서윤이 겪는 현실 필터 누출을 더 강한 물증과 경고로 확장하는 회차입니다.

## 회차 핵심
- 1화는 지하철에서 바로 시작하지 않고, 방송국 회의실에서 프롤로그의 파일을 여는 장면으로 시작한다.
- 폐역 CCTV의 `02:17` 오류와 프롤로그의 횡단보도 프레임 지연이 같은 패턴임을 보여준다.
- 지하철 누출 장면은 중후반 클라이맥스로 재배치한다.
- 마지막에는 승차권 조각과 저장명 없는 번호의 문자로 다음 화 폐역 진입을 예고한다.
